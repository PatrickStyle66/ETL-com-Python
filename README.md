# ETL-com-Python
Desafio realizado para o bootcamp da DIO em parceria com o Santander

# Conceito
A ideia é utilizar uma base de dados disponível no [kaggle](https://www.kaggle.com/datasets/kapturovalexander/bank-credit-scoring/) para captar informações e, com a ajuda de uma IA generativa (chat-gpt), processar essas informações e incrementar a base com novas informações geradas.
Para esta base, o objetivo foi montar um plano de investimento para os usuários de um banco baseado em informações como estado civil, saldo bancário e se o usuário possui algum vínculo empregativo.
